# PSD and ERD/ERS Methodology

## Stage 1 — Per-Epoch Spectral Estimation

For every epoch (trial) of every class, condition, and subject:

1. Call compute_psd_per_epoch with tmin=-2.0, tmax=0.0 → PSD_baseline(f) — shape (n_channels, n_freqs)
2. Call compute_psd_per_epoch with tmin=0.0, tmax=3.0 → PSD_active(f) — same shape
3. Compute the frequency-resolved ERD/ERS:

$ERD/ERS(f) = 10 × log10( PSD_active(f) / PSD_baseline(f) )$

Output: one PSD_baseline, one PSD_active, and one ERD/ERS curve per epoch per channel.

## Stage 2 — Per-Epoch Metric Extraction

For each epoch, each channel, and each band (alpha: 8–13 Hz, beta: 13–30 Hz), extract from both PSD_baseline and PSD_active:

| Metric | Formula | Extracted from |
| --- | --- | --- |
| AUC | trapz(PSD, f) over band | baseline and active |
| Median frequency | f where cumulative sum = 50% of band power | baseline and active |
| Max power | max(PSD[f_low:f_high]) | baseline and active |

Then compute the band-specific ERD/ERS scalar (from the integrated powers, not the curve mean):

$ERD/ERS_band = 10 × log10( AUC_active / AUC_baseline )$

This is more principled than averaging the ERD/ERS(f) curve over the band, because it captures the ratio of total band energy.

**Per epoch you store:**

- 3 metrics × 2 windows (baseline, active) × 5 channels × 2 bands = 60 scalars
- 1 ERD/ERS scalar × 5 channels × 2 bands = 10 scalars
- ERD/ERS(f) curve: (n_channels, n_freqs) — kept for visualization

## Stage 3 — Per-Class Aggregation (within subject)

Average across the ~60 epochs of each class, for each condition:

- Mean and SD of every scalar metric → one representative value per metric per class per subject
- Mean ERD/ERS(f) curve across epochs → visualization of spectral change profile
- Mean PSD_baseline(f) and PSD_active(f) → visualization of absolute spectra

The SD (or SEM) across epochs is the within-subject variability and determines the reliability of each subject's estimates.

## Stage 4 — Group-Level Aggregation

Average per-class means across the 14 subjects:

- Grand mean ± SEM per class, condition, channel, band
- These feed directly into figures: group-average ERD/ERS(f) curves, bar plots per band

## Stage 5 — Statistical Analysis

Three distinct comparisons, each with its own purpose:

### 5.1 — Baseline Sanity Check

Question: Are baseline spectral profiles equivalent across classes and conditions before the cue?

Metrics: AUC_baseline, median_freq_baseline, max_power_baseline

Test: Friedman test (nonparametric one-way repeated-measures) across 7 classes per subject, then across subjects. If no significant difference → baselines are homogeneous → subsequent task comparisons are valid. If differences exist → anticipatory effects or session ordering artifacts → must be reported.

This is not a primary result but a validity check that legitimizes the within-trial ERD/ERS normalization.

### 5.2 — Movement vs. Rest (within condition)

Question: Does each movement class produce significant cortical activation relative to the rest class?

Metrics: AUC_active, ERD/ERS_band (both alpha and beta, both ME and MI)

Test: Wilcoxon signed-rank (paired across 14 subjects) for each movement class vs. rest class

Multiple comparison correction: FDR Benjamini-Hochberg across 6 movement classes × 5 channels × 2 bands

Expected result: Negative ERD/ERS (alpha and beta suppression) for movement classes at C3/C4; rest class ERD/ERS ≈ 0 dB (negative control, validates the metric)

### 5.3 — ME vs. MI (paired, same movement class)

Question: Does motor execution produce stronger or spectrally different cortical activation than motor imagery?

Metrics: AUC_active, ERD/ERS_band, median_freq_active

Test: Wilcoxon signed-rank (paired: each subject contributes one ME value and one MI value for each class)

Correction: FDR BH across 6 classes × 5 channels × 2 bands

Expected result: ME shows stronger ERD (more negative) than MI; topographic distribution may differ

## Data Structure

For each subject, the per-epoch output is organized as:

```
subject → condition (ME/MI) → class → {
    PSD_baseline:      array(n_epochs, n_channels, n_freqs)
    PSD_active:        array(n_epochs, n_channels, n_freqs)
    ERDS_curve:        array(n_epochs, n_channels, n_freqs)

    # Scalars — shape (n_epochs, n_channels, n_bands)
    AUC_baseline:      array(n_epochs, n_channels, n_bands)
    AUC_active:        array(n_epochs, n_channels, n_bands)
    median_freq_base:  array(n_epochs, n_channels, n_bands)
    median_freq_act:   array(n_epochs, n_channels, n_bands)
    max_power_base:    array(n_epochs, n_channels, n_bands)
    max_power_act:     array(n_epochs, n_channels, n_bands)
    ERDS_band:         array(n_epochs, n_channels, n_bands)
}
```

Each subject's processed result is saved to disk (pickle or HDF5). Group-level analysis loads all 14 files and stacks along the subject axis.

Here's the complete structure of each saved pickle:

```
Recording (dict)
│
├── psd_baseline : np.ndarray   shape=(n_epochs, n_channels, n_freqs)   # µV²/Hz, baseline window (−2→0 s)
```

Dimension reference for this dataset:

| Symbol | Value | Source |
| --- | --- | --- |
| n_epochs | variable (~per recording, after rejection) | mixes all 7 classes; split via events[:, 2] |
| n_channels | 5 | C3, C1, Cz, C2, C4 |
| n_freqs | ≈89 | 1–45 Hz at 0.5 Hz grid |
| n_freqs_alpha | ≈11 | 8–13 Hz inclusive |
| n_freqs_beta | ≈35 | 13–30 Hz inclusive |

Note on epoch ordering: n_epochs is the same axis across every array in the file — row i of psd_baseline, erds_curve, metrics['alpha']['active']['auc'], and events all refer to the same trial. Use events[:, 2] (the class code) to group rows by class during the per-class aggregation stage.

## Summary of What Each Metric Contributes

| Metric | Window | Answers |
| --- | --- | --- |
| AUC_baseline | Pre-cue | Baseline power level — sanity check and ERD/ERS denominator |
| AUC_active | Task | Absolute spectral energy during the movement |
| Median_freq_active | Task | Center of mass of spectral energy — frequency shift between classes |
| Max_power_active | Task | Dominant frequency contribution — spectral peak characterization |
| ERD/ERS_band | Both (ratio) | Within-trial relative activation — primary comparison metric |
| ERD/ERS(f) curve | Both (ratio) | Full spectral profile of cortical change — primary visualization |
